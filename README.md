## Requirement
- Ruby 2.4.2 or above
- Jekyll 3.7 or above

## Development

```console
bundle install
bundle exec jekyll serve --livereload
```

open to http://localhost:4000/ for your browser

## Deployment

- This site is hosting to [Netlify](https://www.netlify.com/)
  see also [Docs](https://www.netlify.com/docs/continuous-deployment/)
